# Frontend

PROVIDES frontend design skills for creating distinctive, production-grade interfaces
SO THAT frontend developers using Claude Code
CAN produce interfaces that follow established design principles and accessibility standards

The frontend plugin contains `/designing-frontend` for creating UI components and layouts.

## Assertions

### Compliance

- ALWAYS: prioritize accessibility and semantic HTML in generated interfaces ([review])
- NEVER: produce generic template-like designs — interfaces are distinctive and purpose-built ([review])
