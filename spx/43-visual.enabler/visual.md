# Visual

PROVIDES visual diagramming skills for creating Excalidraw diagrams
SO THAT developers using Claude Code
CAN generate architectural and flow diagrams directly from conversation context

The visual plugin contains `/excalidrawing` for creating Excalidraw-format diagrams.

## Assertions

### Compliance

- ALWAYS: output valid Excalidraw JSON format — diagrams are consumed by the Excalidraw renderer ([review])
